1. Estructura Semántica de HTML5

Las etiquetas semánticas le dicen al navegador (y a Google) qué función tiene cada parte de la página. No son solo cajas: tienen significado.

Etiqueta Para qué sirve Ejemplo de uso
<header> Cabecera de la página o de una sección Logo, título, menú de navegación
<nav> Bloque de navegación con enlaces Menú principal, menú de pie
<main> Contenido principal (solo uno por página) El cuerpo central de la web
<section> Sección temática del contenido "Sobre nosotros", "Servicios"
<article> Contenido independiente y reutilizable Una entrada de blog, una tarjeta de producto
<aside> Contenido secundario o relacionado Barra lateral, publicidad, widget
<footer> Pie de página o de una sección Copyright, enlaces legales, redes
<div> Contenedor genérico SIN significado (no semántico) Agrupar elementos para aplicar CSS

Así se ve la estructura de una página típica:

<header> — Cabecera
<nav> — Navegación
<main> — Contenido principal
<aside> — Barra lateral
<footer> — Pie de página

2. Etiquetas de Texto y Cabeceras

Cabeceras (Headings): h1 al h6

Solo debe haber UN <h1> por página. Los demás se usan en orden descendente.

h1 — Título principal (el más grande)

h2 — Subtítulo de sección

h3 — Subtítulo de subsección

h4 — Nivel 4

h5 — Nivel 5
h6 — Nivel 6 (el más pequeño)

Etiquetas de texto importantes:

<strong> — Texto importante en negrita (tiene significado semántico).

<em> — Texto con énfasis en cursiva (tiene significado semántico).

<b> — Solo negrita visual, sin significado semántico especial.

<i> — Solo cursiva visual, sin significado semántico especial.

<mark> — Texto resaltado como con marcador.

<del> — Texto tachado (borrado).

<ins> — Texto subrayado (añadido).

<small> — Texto más pequeño (notas al pie).

<code> — Para mostrar código informático.

<br> — Salto de línea

4. Enlaces e Imágenes

Etiqueta <a> — Enlace (Anchor)

Enlace externo (abre en nueva pestaña) — usa target="_blank"

Enlace interno a sección — usa href="#id"

Enlace a otro archivo HTML

Enlace de email — usa href="mailto:..."

Enlace de teléfono — usa href="tel:..."

Atributos importantes de <a>:

  • href — La URL de destino (obligatorio)
  • target="_blank" — Abre en nueva pestaña
  • title — Texto emergente al pasar el ratón

Etiqueta <img> — Imagen

La etiqueta <img> es de cierre automático (no necesita </img>).

<img src="ruta/imagen.jpg" alt="Descripción de la imagen" width="300">
  • src — La ruta de la imagen (obligatorio)
  • alt — Texto alternativo si la imagen no carga. ¡Muy importante para accesibilidad!
  • width / height — Dimensiones en píxeles

5. Tablas HTML

Estructura de una tabla completa:

<table>
  <thead>                    ← Cabecera de la tabla
    <tr>                     ← Fila (Table Row)
      <th>Nombre</th>        ← Celda cabecera (Table Header)
      <th>Edad</th>
    </tr>
  </thead>
  <tbody>                    ← Cuerpo de la tabla
    <tr>
      <td>Ana</td>           ← Celda normal (Table Data)
      <td>25</td>
    </tr>
  </tbody>
  <tfoot>                    ← Pie de la tabla (opcional)
    <tr>
      <td colspan="2">Total: 1 persona</td>
    </tr>
  </tfoot>
</table>

Resultado visual:

Nombre Edad Ciudad
Ana García 25 Madrid
Carlos López 30 Barcelona
Total: 2 personas

Atributos de celda: colspan (ocupa N columnas) y rowspan (ocupa N filas).

6. Formularios HTML

Los formularios permiten al usuario enviar datos. La etiqueta <form> los contiene todos.

Atributos de <form>: action (a dónde se envían los datos) y method (GET o POST).

Atributos comunes de inputs:

  • required — Campo obligatorio
  • placeholder — Texto de ayuda dentro del campo
  • disabled — Campo desactivado
  • readonly — Solo lectura, no se puede editar
  • name — Nombre del campo al enviarlo (obligatorio para que se envíe)
  • id — Para enlazarlo con su <label for="...">
  • value — Valor por defecto o valor a enviar

7. Etiquetas Inline vs Block

Etiquetas de bloque (block):

Ocupan toda la línea. Van una debajo de la otra.

<div> <p> <h1>-<h6> <ul> <ol> <li> <table> <form> <section> <header> <footer> <main> <article> <aside> <nav> <blockquote> <pre>

Etiquetas inline:

Solo ocupan su propio espacio. Van una al lado de la otra.

<span> <a> <strong> <em> <img> <input> <label> <button> <code> <mark> <small> <br>
