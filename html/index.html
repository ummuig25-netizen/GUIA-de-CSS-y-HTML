⚡ Resumen Rápido — Conceptos Clave

Selectores: etiqueta / .clase / #id
Box Model: margin > border > padding > content
display: block / inline / inline-block / none
Flexbox: display:flex + justify-content + align-items
Grid: display:grid + grid-template-columns + gap
position: static / relative / absolute / fixed / sticky
z-index necesita position ≠ static
overflow: visible / hidden / scroll / auto
::before y ::after necesitan content: ""
Especificidad: !important > inline > #id > .clase > etiqueta
Selectores: div p / div>p / h2+p / h2~p
Unidades: px / % / em / rem / vh / vw
@keyframes + animation: nombre duración ease infinite
@media (max-width: 600px) { ... }
transition: all 0.3s ease
box-shadow: X Y blur color
Herencia: color y font SÍ / border y margin NO
box-sizing: border-box → padding no agranda la caja
