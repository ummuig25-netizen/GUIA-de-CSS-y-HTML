La Guía Definitiva de CSS 📘

Un manual completo para entender cómo funciona CSS de principio a fin.

1. ¿Cómo seleccionamos elementos? (Los Selectores)

Para darle estilo a algo en HTML, primero CSS necesita saber a quién. Así se hace:

Soy una etiqueta <p> normal. CSS me selecciona solo escribiendo p { ... }

Yo tengo una clase (class="mi-clase"). CSS me selecciona con un punto: .mi-clase { ... }

Yo tengo un ID (id="mi-id"). CSS me selecciona con un numeral: #mi-id { ... }

2. El Modelo de Caja (Box Model)

Todo en la web es un cuadrado/rectángulo. Todo elemento tiene:

  • Content (Contenido): El texto o imagen real.
  • Padding (Relleno): Espacio transparente DENTRO del borde.
  • Border (Borde): La línea que rodea al relleno.
  • Margin (Margen): Espacio transparente FUERA del borde (separa de otros elementos).

6. Tipos de Display: block, inline e inline-block

La propiedad display controla cómo se comporta un elemento en la página:

  • block: Ocupa TODA la línea. El siguiente elemento va debajo. Ej: <div>, <p>, <h1>
  • inline: Solo ocupa lo que necesita. No puedes ponerle width/height. Ej: <span>, <a>
  • inline-block: Lo mejor de los dos: se pone al lado de otros, pero SÍ acepta width y height.
  • none: El elemento desaparece completamente (no ocupa espacio).

7. Overflow (Qué pasa cuando el contenido no cabe)

Cuando el contenido es más grande que su caja, overflow decide qué hacer:

  • visible (default): El contenido se sale de la caja.
  • hidden: Lo que sobra se corta y se oculta.
  • scroll: Siempre aparece una barra de scroll.
  • auto: Solo aparece barra de scroll si es necesario.
visible: El texto se sale de la caja si es muy largo y no cabe dentro de los límites definidos.
hidden: El texto se corta si es muy largo y no cabe dentro de los límites definidos.
scroll: El texto se puede ver haciendo scroll si es muy largo y no cabe dentro de los límites definidos por el CSS.

8. Posicionamiento Avanzado: fixed, sticky y z-index

  • position: fixed — El elemento se queda fijo en la pantalla aunque hagas scroll. Ej: una barra de navegación.
  • position: sticky — Se comporta normal hasta que llegas a él haciendo scroll; entonces se queda pegado.
  • z-index — Controla qué elemento va "encima" de otro (como capas). Mayor número = más arriba.

💡 Nota: Para que z-index funcione, el elemento debe tener position distinto de static (relative, absolute, fixed o sticky).

11. Selectores Combinados (Relacionales)

Podemos combinar selectores para apuntar a elementos muy específicos:

  • div p — Descendiente: Todos los <p> que estén DENTRO de un <div> (a cualquier nivel).
  • div > p — Hijo directo: Solo los <p> que son hijos DIRECTOS del <div>.
  • h2 + p — Hermano adyacente: El primer <p> que va justo DESPUÉS del <h2>.
  • h2 ~ p — Hermanos generales: TODOS los <p> que van después del <h2> al mismo nivel.

12. Unidades CSS y Efectos de Sombra

Unidades de medida:

  • px — Píxeles. Fijo. El más común.
  • % — Relativo al elemento PADRE.
  • em — Relativo al font-size del elemento padre. Si el padre tiene 16px, 2em = 32px.
  • rem — Relativo al font-size del elemento raíz (<html>). Más predecible que em.
  • vh — 1% del alto de la pantalla (Viewport Height). 100vh = pantalla completa en alto.
  • vw — 1% del ancho de la pantalla (Viewport Width). 100vw = pantalla completa en ancho.
